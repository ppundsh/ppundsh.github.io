刷 Merlin

另外，N66U 的處理器是 Broadcom BCM4706，處理器架構是 MIPS 74K1，可用下列指令查看

安裝 KMS Server ： vlmcsd

介紹2

常用的 KMS 激活服務器程序為 py-kms 與 vlmcsd，分別為 python 和 C 語言版本。可運行於各種類 linux 和 windows 及 mac os 環境中。

開啟 jffs

  1. 開啟 jffs 和 ssh ，登入 n66u 管理介面，在 系統管理→系統設定，開啟 Format JFFS partition at next boot 和 Enable JFFS custom scripts and configs
  2. 開啟 Enable SSH ，選 LAN only
  3. 重開路由器
  4. 進入管理介面，把 Format JFFS partition at next boot 取消

安裝 vlmcsd

  1. 在官網下載 vlmcsd ，複製出執行檔
    mips 架構的路由器使用 / binaries/Linux/mips/little-endian/static/vlmcsd-mips32el-uclibc-static
    arm 架構的路由器使用 / binaries/Linux/armlittle-endian/static/vlmcsd-armv7el-uclibc-static
  2. 將文件重命名為 vlmcsd，通過 xftp，winscp 等工具(連線類型還SCP，端口和 SSH 設定的一樣) 上傳到路由器 / jffs 目錄下

運行命令

腳本自動掃瞄路由器外接的 USB 設備，列出所有存儲類設備供選擇使用哪個 USB 硬盤作為虛擬記憶體的載體。
支持 NTFS\EXT2,3,4
注意：建議使用高速隨身碟或者 SSD 硬碟，虛擬記憶體類似於普通記憶體，需要多次高速讀寫，整體速度取決於 USB 設備的速度。這裡可以選擇虛擬記憶體的大小，默認為 33000=32M，可以設置為 64M=>65500。如果沒有特殊需求，建議使用默認大小。並不是越大越好，更大的虛擬記憶體容量 可能會拖慢系統速度。

加載成功後，自動加載 Swap 的信息。Total 是總緩存，Used 已使用，Free 未使用
如果 Total 顯示為 0 證明沒有加載成功或者已關閉虛擬記憶體功能。
最後一行是提示是否加載到開機啟動項。y 回車是添加到啟動項。
