剛好遇到，小記一下。
用開機光碟修改，以便在鎖定畫面叫出 cmd 改密碼。

用開機光碟，不要按下一步，用指令叫出 cmd

用 diskpart 查看分區

將螢幕小鍵盤改成 cmd

或

(登入界面按五次 shift )

重開機後，點右下角的“輕鬆讀取”，點螢幕小鍵盤，用 net user 改密碼的指令

完成後按 exit 離開。
